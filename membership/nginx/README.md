# 🧩 NGINX-PHP-MySQL 연동 방법

## NGINX 서버 실행

nginx.exe 실행 (다운로드 받은 폴더에 존재)

> 프로세스 실행 중인지 확인  
> 또는 localhost 접속해보기

## PHP 실행

`cmd` 실행 → PHP 다운로드 받은 경로로 이동  
```bash
php-cgi -b 127.0.0.1:9000
```
명령어 실행으로 php 연동 

## MySQL 연동

`MySQL WorkBranch` 또는 `Heidisql`로 Local DB 설정


IDE인 `phpstorm`으로 Database 연동하면 편함!
